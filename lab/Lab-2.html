Splitting Dataset in Training and Testing Dataframes

In order to make it easier to analyse our dataset it is important we split our dataset into two uneven proportions (for example 30% for training and 70% for testing)

We will work with the training dataset in order for our computers to run our algorithms quicker. Upon completion of our models, we can then visualize with our full dataset

For now we will split our dataset into a random training dataframe (which is made up of 10% of the main dataset):

n = nrow(df)

Plotting with ggplot2

ggplot2 is a system for declaratively creating graphics, based on The Grammar of Graphics. You provide the data, tell ggplot2 how to map variables to aesthetics, what graphical primitives to use, and it takes care of the details. (Source https://ggplot2.tidyverse.org/index.html )

To install the package:

#install.packages("ggplot") 

#OR we can install it by installing the TidyVerse package
#install.packages("tidyverse")

Download the ggplot2 Cheat Sheet here: https://github.com/rstudio/cheatsheets/raw/master/data-visualization-2.1.pdf

Load the package and begin plotting

library(ggplot2)
#A sample plot for first 10 of our training dataset
ggplot(df_train[1:10,], aes(date,gold))+
  geom_bar(stat = "identity")

#we can also plot points
ggplot(df_train[1:10,]) + 
  geom_point(aes(date,gold))

#full plot of gold stats
ggplot(df_train, aes(date,gold))+
  geom_bar(stat = "identity")

By default, geom_ bar uses stat=count which makes the height of the bar proportion to the number of cases in each group (or if the weight aethetic is supplied, the sum of the weights). If you want the heights of the bars to represent values in the data, use stat=identity and map a variable to the y aesthetic.

#plot platinum stats 
ggplot(df_train, aes(date,platinum)) +
  geom_point(aes(colour = platinum)) #replace colour = platinum ----> colour = "platinum" 

ggplot(df_train, aes(USD_840, colour = silver)) +
  geom_freqpoly(binwidth = 10)

Exploratory Data Analysis (EDA)

EDA is to develop an understanding of your data. Questions are used as tools to guide the process. There is no rule about which questions you should ask to guide your research. Source: https://r4ds.had.co.nz/exploratory-data-analysis.html#introduction-3
